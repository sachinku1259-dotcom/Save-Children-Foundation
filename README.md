# Save Children Donation Website

A simple and responsive donation website built using HTML and CSS to help raise funds for poor children suffering from cancer and serious illnesses.

## Features

* Attractive Home Page
* Child Illness & Cancer Awareness Section
* Donation Section
* QR Code Payment Support
* UPI Payment Details
* Bank Account Details
* Child Gallery
* Contact Information
* Mobile Responsive Design
* Smooth Scrolling Navigation

## Project Structure

```text
save-children/
│
├── index.html
├── style.css
│
└── images/
    ├── qr.png
    ├── child1.jpg
    ├── child2.jpg
    └── child3.jpg
```

## Installation

1. Download or clone the project.
2. Place your QR code inside the images folder as `qr.png`.
3. Add child images inside the images folder.
4. Update your UPI ID, Account Number, and IFSC Code in `index.html`.
5. Open `index.html` in your browser.

## Donation Information

Replace the following with your details:

* UPI ID
* Account Name
* Account Number
* IFSC Code
